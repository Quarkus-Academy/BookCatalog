# Book Catalog (Quarkus Academy)

## Running the application

### Development

```shell script
quarkus dev
```
or
```shell script
./mvnw quarkus:dev
```

### Testing

```shell script
quarkus test
```
or
```shell script
./mvnw quarkus:test
```
